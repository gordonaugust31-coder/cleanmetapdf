# 🛡️ CleanMeta

Заміна та очистка метаданих PDF, DOCX, XLSX, PPTX, ODT — захист конфіденційності перед відправкою файлів.

## Що робить

**PDF** — підставляє реалістичні корпоративні метадані (SAP, Oracle, IBM тощо — 19 варіантів)

**Office/ODT** — повна очистка: автор, компанія, програма, дата редагування

## Запуск локально

```bash
brew install exiftool
pip install streamlit
streamlit run cleanmeta_app.py
```
